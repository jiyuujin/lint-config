# @nekohack/stylelint

## Installation

Install the StyleLint plugin for CSS.

```bash
npm i @nekohack/stylelint-config -D
yarn add -D @nekohack/stylelint-config
```

### Configuration

You can use as general StyleLint plugins.

```js
module.exports = {
    extends: ['@nekohack/stylelint-config']
}
```
